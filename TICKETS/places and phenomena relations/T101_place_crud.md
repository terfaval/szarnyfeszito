# T101 – Place CRUD (Core Entity)

## Goal
CRUD-lite UI for Place entity (admin-only).

## Scope
- Places list page (search)
- Create/edit Place fields:
  - name
  - slug
  - region
  - place_type
  - status (draft/published later)
- No map UI yet (location fields optional)

## Acceptance Criteria
- Place create/edit persists
- List shows saved places
