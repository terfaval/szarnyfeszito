# Patch Library -- ACL-barát Jóga + Erősítő Rendszer

Ez a dokumentum egy **patch-alapú edzéskönyvtár**, amelyből jóga és
futás mellé rugalmasan lehet blokkokat kombinálni.

A patch-ek két típusba tartoznak:

1.  **Teljes testes erősítő patch-ek**
    -   könnyű
    -   intenzív
2.  **Funkcionális cél-patch-ek**
    -   stabilitás
    -   technika
    -   testtartás

Minden patch rövid (15--30 perc), így könnyen kombinálható jógával.

------------------------------------------------------------------------
